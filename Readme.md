# Frontend Machine Coding

A collection of small frontend machine-coding projects focused on clean React architecture, quick problem-solving, and practical UI components. This repo serves as a lightweight space to sharpen real-world frontend skills through hands-on mini projects.

## Features
- Fast, self-contained micro-projects  
- Clean and readable React code  
- Practical UI components used in real apps  
- Great for interview prep and frontend practice  

## Tech Stack
- React  
- Vite  
- JavaScript 
- CSS / Tailwind (depending on the component)

## Project Structure
Each mini-project lives in its own folder with isolated logic and styling, making it easy to explore, modify, or extend.

## How to Run
```
npm install
npm run dev
```
